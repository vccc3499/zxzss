# Автоассистент откликов (Сочи, от 150000)

Файл: `job_hunter_assistant.py`

## Что делает
- Ищет вакансии через HH API по Сочи.
- Фильтрует по зарплате от 150000.
- Генерирует черновики откликов в `outbox_replies.jsonl`.
- Генерирует ответ работодателю по входящему сообщению.
- Если распознано приглашение на собеседование, отправляет уведомление в Telegram (дата, время, номер телефона).

## Важное ограничение по Avito
- Browser-автоматизация Avito отключена (safe mode), чтобы не ловить антибот/IP-циклы.
- Рабочая схема: вы вручную копируете входящее сообщение работодателя в команду `reply-chat`.

## Переменные окружения
- `OLLAMA_MODEL` — локальная модель (если пусто, используется OpenRouter).
- `OPENROUTER_API_KEY` — ключ OpenRouter (fallback, если нет локальной модели).
- `OPENROUTER_MODEL` — модель OpenRouter.
- `TELEGRAM_BOT_TOKEN` — токен бота Telegram.
- `TELEGRAM_CHAT_ID` — ваш chat id в Telegram.

## Быстрый старт

1. Проверить вакансии:
```powershell
py job_hunter_assistant.py scan --city "Сочи" --min-salary 150000
```

2. Сгенерировать отклики:
```powershell
py job_hunter_assistant.py auto --city "Сочи" --min-salary 150000 --profile "Я дисциплинированный и старательный, быстро обучаюсь, готов работать и развиваться."
```

3. Обработать сообщение работодателя вручную (без спама в старые чаты):
```powershell
py job_hunter_assistant.py reply-chat --incoming "Приглашаем на собеседование 12.03 в 11:00, +7 900 111-22-33" --profile "Я старательный, быстро обучаюсь, готов оперативно выйти на работу." --employer "ООО Ромашка" --vacancy "Менеджер" --notify-telegram
```

## Файлы состояния
- `job_state.json` — обработанные вакансии и уже отправленные интервью-уведомления.
- `outbox_replies.jsonl` — черновики откликов.
- `chat_replies.jsonl` — история входящих сообщений и ответов.
